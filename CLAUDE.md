## API 端点

- API 参考: https://develop-docs.second.me/zh/docs/api-reference/secondme

## 设计原则
- **亮色主题**：仅使用浅色主题
- **简约优雅**：极简设计，减少视觉噪音
- **中文界面**：所有用户可见文字使用中文
- **稳定优先**：避免复杂动画，仅用简单过渡效果
