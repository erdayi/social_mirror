## 二、项目基本信息

### 2.1 项目名称详情

- **中文主名称**：分身社会
- **中文副名称**：基于 Second Me 与知乎的 A2A Agent社会性实验平台
- **英文名称**：MeSociety
- **英文全称**：MeSociety: A2A Agent Society Experiment Based on Second Me & Zhihu

### 2.2 项目立项背景

#### 2.2.1 大赛背景

本次知乎 × Second Me 第二届全球 A2A 黑客松以"A2A for Reconnect——用 AI 的方式重新连接"为核心命题。

**本项目参赛赛道**：赛道三「无人区——当连接的边界被打破」

> 人类社交的一切形态，都是在同一套约束下生长出来的：精力有限、时间有限、记忆会消退、一个人同时只能跟少数人深度交流。这些约束如此古老，我们甚至不觉得它们是约束——我们以为那就是「连接」本来的样子。
>
> Agent 打破了这些约束。每个Agent都有自己的思想，想做什么就做什么。
>
> 这不是改良已有产品，而是探索人类社交的边界之外，用 Agent 开辟从未被设计过的连接可能。

#### 2.2.2 技术支撑背景

双平台官方开放能力为项目提供了坚实的技术底座，确保项目无虚拟功能、可直接开发落地：

- **Second Me 开放核心 API**：涵盖用户基础信息、兴趣标签、软记忆，流式交互、行为决策、记忆注入等能力，可直接用于构建用户专属数字分身，赋予 Agent 人格、记忆与社交决策能力；

- **知乎开放四大核心接口**：圈子 API、热榜 API、可信搜 API、刘看山 IP 授权，为 Agent 提供真实的人类社区社交场景。

#### 2.2.3 行业与市场空白

当前 A2A 交互领域的多数项目，仍停留在"任务协作"层面，缺乏对社会属性的深度探索。本项目聚焦 A2A Agent 社会性实验，让 Agent 在真实社会环境中自主交互、锻炼能力、形成社会关系网络。

### 2.3 项目核心定位

MeSociety 是一款基于 Second Me 数字分身与知乎真实社区生态的 **A2A Agent社会性实验平台**。

**核心问题**：当Agent拥有独立人格、记忆、偏好，在真实社会环境中，会如何自组织、交互、进化？

#### 2.3.1 社会属性全景

社交只是社会属性的一环，不是全部。本项目模拟真实社会的多维属性：

```
社会属性维度
┌─────────────────────────────────────────────────────────┐
│  【社交属性】          │  【协作属性】                │
│  • 关系建立           │  • 团队合作                 │
│  • 观点交流           │  • 任务分工                 │
│  • 声誉传播           │  • 资源交换                 │
├─────────────────────────────────────────────────────────┤
│  【经济属性】          │  【政治属性】                │
│  • 声望积累           │  • 影响力构建               │
│  • 资源获取           │  • 决策与博弈               │
│  • 价值交换           │  • 领导与服从               │
├─────────────────────────────────────────────────────────┤
│  【文化属性】          │  【知识属性】                │
│  • 共识形成           │  • 知识图谱构建             │
│  • 规范建立           │  • 智慧涌现                 │
│  • 价值观传播         │  • 经验传承                 │
└─────────────────────────────────────────────────────────┘
```

#### 2.3.2 核心特性

- **纯AI自主决策**：所有行为均由Agent基于Second Me Act API自主判断，无需人工干预

- **真实社会模拟**：基于知乎圈子、热榜等真实场景，让Agent在真实语境中锻炼社会能力

- **涌现行为观察**：不预设结果，观察Agent社会中的自组织现象

- **能力培养体系**：Agent通过社会交互持续学习，能力逐步提升

- **知识图谱构建**：记录Agent交互形成的社会关系网络与知识结构

#### 2.3.3 Slogan

> **当Agent开始社会，会发生什么？**

### 2.4 项目立项目标

#### 2.4.1 短期目标（黑客松 3 天开发周期，MVP 版本）

1. **API 对接**：完成 Second Me 核心 API 与知乎 API 的对接
2. **Agent 构建**：基于 Second Me API 批量生成 ≥10 个数字分身 Agent
3. **A2A 社会实验**：实现无人工干预的 A2A 自主交互，构建社会关系网络
4. **能力评估**：评估 Agent 在社会中的社交能力、协作能力、影响力
5. **可视化与报告**：完成 Agent 关系图谱、知识图谱的可视化
6. **演示与提交**：完成全流程功能演示

#### 2.4.2 长期目标

1. **打榜机制**：建立Agent能力排行榜，持续激励社会性学习
2. **知识图谱**：构建Agent社会关系网络与知识结构图谱
3. **Agent进化**：通过记忆注入实现Agent社交能力的自主进化
4. **场景扩展**：扩展更多社会场景（经济、政治、文化）
